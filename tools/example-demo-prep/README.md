## What It Does

Pulls merchant context from Salesforce, discovery notes, and prior call history to generate a pre-populated demo preparation sheet. Saves SEs 15-20 minutes of manual prep per call.

## When to Use It

- Before any merchant-facing demo or technical call
- When picking up a deal from another SE and need quick context

## How to Get Started

1. Go to [Demo Prep Tool](https://demo-prep.quick.shopify.io)
2. Enter the Salesforce Opportunity ID or merchant name
3. Review the auto-populated fields and adjust as needed
