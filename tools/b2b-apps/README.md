# B2B App Directory

A searchable directory of app guidance for B2B solutioning. It adds an SE decision layer on top of the [B2B App List spreadsheet](https://docs.google.com/spreadsheets/d/1KEf54Wnnpj93KEkBUiSVwjCdao57ERvPmPB84zJyhrY/) so apps are not just discoverable, but easier to position.

## Coverage

- **Shopify Native** — Flow, Forms, Search & Discovery, Combined Listings, Bundles, Checkout Blocks
- **B2B Operations** — Bulk order, CSV upload, quoting, sales rep tools, company management
- **ERP & Integrations** — NetSuite, Dynamics 365, Pipe17, PIM connectors, accounting sync
- **Invoicing & Quoting** — Sufio, Order Printer Pro, SA Request a Quote
- **Specialty** — Punchout, subscriptions, shipping, discounts, loyalty, metafields, and more

## Guidance Tiers

- **Native-first** — start with Shopify primitives before introducing another dependency
- **SE-vetted** — useful in common SE patterns, with requirement fit still validated
- **Caveats** — viable, but needs extra scrutiny around fit, UX, data, or ownership
- **Partner-led** — implementation or integration decisions that should involve the right partner or technical owner
- **Avoid / legacy** — do not lead with these for B2B unless there is a scoped reason
