# B2B App Directory

A searchable directory of SE-vetted apps for B2B solutioning. Sourced from the [B2B App List spreadsheet](https://docs.google.com/spreadsheets/d/1KEf54Wnnpj93KEkBUiSVwjCdao57ERvPmPB84zJyhrY/).

## Coverage

- **Shopify Native** — Flow, Forms, Search & Discovery, Combined Listings, Bundles, Checkout Blocks
- **B2B Operations** — Bulk order, CSV upload, quoting, sales rep tools, company management
- **ERP & Integrations** — NetSuite, Dynamics 365, Pipe17, PIM connectors, accounting sync
- **Invoicing & Quoting** — Sufio, Order Printer Pro, SA Request a Quote
- **Specialty** — Punchout, subscriptions, shipping, discounts, loyalty, metafields, and more
