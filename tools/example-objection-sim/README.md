## What It Does

Generates realistic merchant objection scenarios based on common competitive situations (BigCommerce, Adobe Commerce, Salesforce Commerce Cloud). Provides real-time coaching feedback using the S.T.A.R. objection handling framework.

## When to Use It

- Preparing for a competitive deal where you expect objections
- Ramping up new SEs on competitive positioning
- Team training sessions and enablement

## How to Get Started

1. Go to [Objection Handling Simulator](https://objection-sim.quick.shopify.io)
2. Select a competitor and scenario type
3. Respond to the AI merchant and receive coaching feedback
