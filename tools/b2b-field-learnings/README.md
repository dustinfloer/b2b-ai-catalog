# B2B Field Learnings

A searchable knowledge base of real-world B2B learnings from the SE team — the edge cases, API patterns, and workarounds that aren't in the official docs.

## Coverage

- **Payments** — Auto capture behavior, terms + card requirements, VPF determination, Tap to Pay
- **APIs & Mutations** — Charging saved cards/ACH, creating B2B orders, store credit, catalog APIs
- **Workflows** — Sales rep order submission, draft order patterns, custom payment terms, restricted products
- **Tax & Compliance** — B2B US tax decision logic (nexus × location settings × delivery address)
- **Integrations** — NetSuite (native + Celigo), SSO/identity providers
- **Platform** — Inventory check timing, themes built for B2B, site monitoring, Plus trials

## Source

All learnings are derived from the [B2B Field Learnings & Solutions Google Doc](https://docs.google.com/document/d/1oBVz0kz3OkkZxCQv2k7UpwfiZx9lYXc_yddP_gdwMB0/edit) — the Google Doc remains the source of truth and is linked from every card.
