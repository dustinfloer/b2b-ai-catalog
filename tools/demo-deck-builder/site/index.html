Install

Drop the skill into your IDE, start building.

Works for anyone on the SE or AE team using Claude Code or Cursor. Clone once, copy the skill, invoke by name — the AI reads your merchant context and builds the deck.

🤖
Claude Code
SEs + AEs · ~60 seconds
1Download the skill (one command, ~30 KB)
mkdir -p ~/.claude/skills/demo-deck-builder && curl -sL https://github.com/dustinfloer/b2b-ai-catalog/archive/main.tar.gz | tar -xz --strip-components=4 -C ~/.claude/skills/demo-deck-builder b2b-ai-catalog-main/tools/demo-deck-builder/skill
2Restart Claude Code, then run
/demo-deck-builder [merchant-name]
Claude reads your merchant folder, asks 3 questions, builds the full deck.
🖱️
Cursor
SEs + AEs · ~60 seconds
1Download the skill (one command, ~30 KB)
mkdir -p ~/.cursor/rules/demo-deck-builder && curl -sL https://github.com/dustinfloer/b2b-ai-catalog/archive/main.tar.gz | tar -xz --strip-components=4 -C ~/.cursor/rules/demo-deck-builder b2b-ai-catalog-main/tools/demo-deck-builder/skill
2In Cursor chat, invoke it
@demo-deck-builder build a deck for [merchant]
Cursor reads the skill, follows the same 7-step workflow as Claude Code.
