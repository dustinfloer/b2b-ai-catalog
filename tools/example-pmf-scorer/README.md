## What It Does

Evaluates opportunities across 10 PMF categories using Gauge's B2B qualification framework. Generates tier scores (0-4), identifies discovery gaps, and suggests follow-up questions to improve qualification.

## When to Use It

- During or after discovery calls to assess deal quality
- When prioritizing pipeline and deciding where to invest SE time
- Before submitting a deal for leadership review

## How to Get Started

1. Go to [PMF Scorer](https://pmf-gauge.quick.shopify.io)
2. Select the opportunity or enter details manually
3. Walk through each category and review the tier scores
